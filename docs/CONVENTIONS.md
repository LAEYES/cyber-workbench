# Conventions (naming, structure)

## Prefixes de livrables
Tous les livrables générés suivent le schéma :

`<PREFIX>_<DomaineXX>_<Sujet>_<ORG>_<LANG>.md`

Exemples :
- `POL_D01_PSSI_ACME_FR.md`
- `STD_D02_MFA_ACME_EN.md`
- `PROC_D07_IncidentResponse_ACME_FR.md`
- `CHK_D09_VulnerabilityManagement_ACME_EN.md`
- `PLB_D15_Ransomware_ACME_FR.md`

### Préfixes
- **POL** : Policy (politique)
- **STD** : Standard (standard/exigences)
- **PROC** : Procedure (procédure)
- **SOP** : Standard Operating Procedure (opérationnel)
- **CHK** : Checklist (contrôle de conformité / audit)
- **PLB** : Playbook (IR/SOC)
- **TMP** : Template (modèle)
- **MAP** : Mapping (matrice / correspondance)

## Langues
- **FR** : français
- **EN** : English
