# STD — D07 — Data Classification Standard

**Organization:** ACME  
**Version:** 0.2 (hardened)  
**Date:** 2026-02-03

## 1. Levels (minimum)
- Public
- Internal
- Confidential
- Highly sensitive

## 2. Rules (pass/fail)
- Each critical dataset/app has an owner.
- Confidential+: public sharing prohibited.
- Highly sensitive: need-to-know access + access logging.

## 3. Labeling
- Label documents and workspaces for Confidential+.

## 4. Review
- Classification review: yearly (baseline), semi-annual for critical scope (regulated).

## 5. Evidence
- Classification matrix + owners + label examples.
