# STD — D07 — Standard Classification des Données

**Organisation :** ACME  
**Version :** 0.2 (durci)  
**Date :** 2026-02-03

## 1. Niveaux (minimum)
- Public
- Interne
- Confidentiel
- Très sensible

## 2. Règles (pass/fail)
- Chaque dataset/app critique a un owner.
- Confidentiel+ : partage public interdit.
- Très sensible : accès restreint (need-to-know) + logs accès.

## 3. Étiquetage
- Étiquetage des documents et espaces (SharePoint/Drive) pour Confidentiel+.

## 4. Revue
- Revue classification : annuelle (baseline), semestrielle (régulé sur périmètre critique).

## 5. Preuves
- Matrice classification + owners + exemples d’étiquettes.
