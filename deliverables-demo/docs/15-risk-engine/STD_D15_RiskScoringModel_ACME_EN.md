# STD — D15 — Risk Scoring Model

**Organization:** ACME  
**Version:** 0.2 (hardened)  
**Date:** 2026-02-03

## 1. Score
- Likelihood (1–5) × Impact (1–5) = Score (1–25)

## 2. Thresholds (example)
- 1–5: monitor
- 6–10: planned
- 11–15: priority
- 16–25: immediate / exec notified

## 3. Rules
- Rationale required for L and I.
- Review score ≥ 16: weekly.

## 4. Evidence
- Risk register + review history.
