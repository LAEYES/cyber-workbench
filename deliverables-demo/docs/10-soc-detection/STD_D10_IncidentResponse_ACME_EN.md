# STD — D10 — Incident Response (IR) Standard

**Organization:** ACME  
**Version:** 0.2 (hardened)  
**Date:** 2026-02-03

## 1. Baseline requirements
- Runbooks: phishing, ransomware, admin account compromise, data breach.
- Containment SLAs (target):
  - Critical: **≤ 4h**
  - High: **≤ 24h**
- IR exercises: **yearly**.

## 2. Enhanced (regulated)
- Exercises: **semi-annually** (critical).
- Legal/compliance coordination (D08) integrated.
- Evidence: timeline + decisions + artifacts.

## 3. Evidence
- Runbooks, exercise reports, IR tickets.
